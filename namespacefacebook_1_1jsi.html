◆ HostFunctionType

using facebook::jsi::HostFunctionType = typedef std::function< Value(Runtime& rt, const Value& thisVal, const Value* args, size_t count)>

A function which has this type can be registered as a function callable from JavaScript using Function::createFromHostFunction(). When the function is called, args will point to the arguments, and count will indicate how many arguments are passed. The function can return a Value to the caller, or throw an exception. If a C++ exception is thrown, a JS Error will be created and thrown into JS; if the C++ exception extends std::exception, the Error's message will be whatever what() returns. Note that it is undefined whether HostFunctions may or may not be called in strict mode; that is thisVal can be any value - it will not necessarily be coerced to an object or or set to the global object.
