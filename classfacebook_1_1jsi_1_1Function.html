Calls the function with any number of arguments similarly to Object::setProperty(). The this value of the JS function will not be set by the C++ caller, similar to calling Function.prototype.call(undefined, ...args) in JS. Note: as with Function.prototype.call, this may not always be undefined in the function itself. If the function is non-strict, this will be set to the global object.

Create a function which, when invoked, calls C++ code. If the function throws an exception, a JS Error will be created and thrown.

Parameters
namethe name property for the function.
paramCountthe length property for the function, which may not be the number of arguments the function is passed.
Note
The std::function's dtor will be called when the GC finalizes this function. As with HostObject, this may be as late as when the Runtime is shut down, and may occur on an arbitrary thread. If the function contains any captured values, you are responsible for ensuring that their destructors are safe to call on any thread.

◆ getHostFunction()

HostFunctionType & facebook::jsi::Function::getHostFunction ( Runtime &  runtime) const
inline

Returns the underlying HostFunctionType iff isHostFunction returns true and asserts otherwise. You can use this to use std::function<>::target to get the object that was passed to create the HostFunctionType.

Note: The reference returned is borrowed from the JS object underlying this, and thus only lasts as long as the object underlying this does.
