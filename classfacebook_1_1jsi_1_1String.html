◆ getStringData()

template<typename CB >
void facebook::jsi::String::getStringData ( Runtime &  runtime,
CB &  cb 
) const
inline

Invokes the user provided callback to process content in String. The callback must take in three arguments: bool ascii, const void* data, and size_t num, respectively. ascii indicates whether the data passed to the callback should be interpreted as a pointer to a sequence of num ASCII characters or UTF16 characters. The function may invoke the callback multiple times, with a different format on each invocation. The callback must not access runtime functionality, as any operation on the runtime may invalidate the data pointers.
